1. 모듈생성

example.js에

- module.exports = 객체( {속성: 값, ...} / new 생성자함수이름(...) );

또는

- module.exports = 함수( function(...){...} );

또는

- module.exports = 배열( [ ... ] ); 등등

-> 다양한 자료형 모듈화, 한 파일에 하나의 기능만

- module.exports.속성 = 객체 / 함수 / 배열 / ... ;

-> 속성으로 구분하여 모듈화, 한 파일에 여러 기능 추가 가능


2. 모듈사용

다른 파일에서

- var example = require(./example);

으로 불러와

example에 담긴 값이
'객체 / 함수 / 배열 / ... '중 하나인 경우

- example

'속성'으로 구분하여 여러 개인 경우

- example[속성]

을 사용


3. 그 외 객체에 대한 복습

객체: 여러 자료형의 모임
(자바스크립트에서는 함수도 자료형으로 분류)

객체의 속성: 각 자료값의 이름

객체의 '속성값' 사용 방법:

- 객체이름.속성;

객체의 '함수 호출' 방법:

- 객체이름.속성(매개변수);
